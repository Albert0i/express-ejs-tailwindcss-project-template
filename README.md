### express-ejs-tailwindcss-project-template 

[Setting up Express MVC + EJS + TailwindCSS (4.0)](https://medium.com/@hannnirin/setting-up-express-mvc-ejs-tailwindcss-4-0-2ccac72dad59)

```
npm init -y

npm install express ejs

npm install -D postcss tailwindcss postcss-cli @tailwindcss/postcss autoprefixer nodemon prettier concurrently

Linux: 
mkdir src
mkdir src/views src/public src/public/styles src/public/views

Windows: 
mkdir src
mkdir src\\views src\\public src\\public\\styles src\\public\\views

npm start

npm run build:css

npm run dev

http://localhost:8081

http://localhost:8081/views/index.html

```
